#  Crypto Terminal Pro - Dashboard Financiero

### 🔗 [Ver Aplicación en Vivo](https://facundoconsultas.github.io/crypto-dashboard/)

Terminal financiera avanzada que consume datos en tiempo real de la API de **CoinGecko** para mostrar precios, tendencias y análisis histórico de criptomonedas en dólares (USD).

##  Características Principales
- ** Gráficos Dinámicos:** Visualización de tendencias de los últimos 7 días utilizando la librería **Chart.js**.
- ** Buscador Global:** Permite consultar estadísticas de cualquier criptomoneda del mercado por nombre.
- ** Datos en USD:** Precios, capitalización de mercado (Market Cap) y volumen de transacciones actualizados.
- ** Interfaz Pro:** Diseño oscuro (Dark Mode) optimizado para analistas, con efectos de hover y feedback visual.
- ** Responsive Design:** Adaptado para una visualización correcta en dispositivos móviles y tablets.

##  Tecnologías Utilizadas
- **HTML5 & CSS3** (Flexbox, Grid y Variables de CSS).
- **JavaScript Moderno (ES6+)**: Uso intensivo de `fetch`, `async/await` y manipulación del DOM.
- **Chart.js**: Para el renderizado de gráficos de línea suaves.
- **CoinGecko API**: Fuente de datos del mercado cripto.
